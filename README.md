# zlui-viewer README

Preview zlUI, default file ext: *.ui

## Features

* Structure

```
Object Panel
{

    # child object
    Object Button
    {
        
    }
}
```

* Object Type

Win

Image extend Win

Panel extend Image

Edit extend Panel

Button extend Panel

Check extend Button

Combo extend Button

Slider extend Panel


## Requirements

UI system rendering base on @zhobo63\imgui-ts
